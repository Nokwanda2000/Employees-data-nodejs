# Employees-App-Task6


# Run Locally

Clone the project
```bash
  git clone https://github.com/Nokwanda2000/Employees-App-Task6.git
```

Go to the project directory
```bash
  cd employees-app-task6
```

Install dependencies
```bash
  npm install
```

Start the react server
```bash
  npm run start
```

## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
